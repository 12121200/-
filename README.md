Вот максимально краткие ответы на все контрольные вопросы (в порядке, как они идут в твоём тексте):

1. **Чем отличается «свернуть» окно от «закрыть» окно?**  
   Свернуть — окно убирается на панель задач, программа продолжает работать. Закрыть — программа полностью завершается.

2. **Для чего нужна панель задач и что значит «закрепить приложение»?**  
   Панель задач — показывает запущенные программы и быстрый доступ. Закрепить — приложение остаётся на панели всегда, даже после закрытия.

3. **Чем отличается «копировать» от «переместить»?**  
   Копировать — создаётся копия, оригинал остаётся. Переместить — файл переносится, оригинал удаляется с прежнего места.

4. **Что такое расширение файла и почему его важно видеть?**  
   Расширение — часть имени после точки (.txt, .jpg и т.д.), определяет тип файла. Видеть важно, чтобы правильно открывать файл и не запускать опасные файлы случайно.

5. **Как восстановить файл из корзины и когда это невозможно?**  
   Открыть Корзину → правой кнопкой по файлу → «Восстановить». Невозможно, если корзина очищена или файл удалён с Shift+Delete.

6. **Чем полезен поиск в Проводнике и какие параметры можно уточнять?**  
   Полезен для быстрого нахождения файлов. Уточнять можно: имя, тип, дату, размер, содержимое, место хранения.

7. **Как быстро переключаться между окнами без мыши?**  
   Alt + Tab.
