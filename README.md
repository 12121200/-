В ходе выполнения практической работы быстро получилось ориентироваться в элементах рабочего стола и управлять окнами, поскольку эти действия уже были знакомы из повседневного использования Windows. Непривычным оказалось детальное изучение свойств файлов и настройка отображения расширений, так как раньше я не обращал на это внимания, и это требовало поиска в параметрах Проводника. Запомнил сочетания клавиш Win + E для открытия Проводника, Alt + Tab для переключения между окнами и Win + Shift + S для создания скриншотов, которые значительно ускорили работу. Я понял, что эффективное использование интерфейса Windows помогает экономить время в учебных задачах.
