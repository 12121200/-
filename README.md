### 1. Распределяющая шляпа Хогвартса 🧙

```python
score = int(input("Введите количество баллов: "))

if score >= 90:
    print("Гриффиндор")
elif score >= 70:
    print("Когтевран")
elif score >= 50:
    print("Пуффендуй")
else:
    print("Слизерин")
```

### 7. Магазин зелий 🧪

```python

else:
    print(f"Скидки нет. К оплате: {price} рублей")
```
